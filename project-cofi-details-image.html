Cofi

Summary

Inspired by SoundCloud and Spotify with a minimalist approach and lofi-genre theme. Cofi is a web music player that allows users to upload, edit, and listen to their own or other user's music.

Technologies

JavaScript, React, Redux, Python, Flask, SQLAlchemy, Alembic, PostgreSQL, Amazon Web Services S3, HTML5, CSS3, Docker
